## 風險門檻系統（2026-05-12 重構）

### 多因子聯鎖進場條件

**加倉美股需同時滿足（Green Zone）：**
1. TWII RSI < 80（必要非充分條件）
2. VIX < 25（市場情緒穩定）
3. WTI 原油 < 95（排除通膨失控）
4. 目標標的 RSI < 75（個股自己不能太貴）
5. 美股趨勢向上（MA20 > MA60）

**任一條件滿足則禁加倉（Red Zone）：**
- ✗ VIX > 25（市場恐慌）
- ✗ TWII RSI > 90（台股明顯過熱）
- ✗ WTI 原油 > 100（原物料通膨失控）
- ✗ 標的 RSI > 80（自己已經超買）

### 信號燈系統

| 指標 | Red Zone（禁） | Yellow Zone（警戒） | Green Zone（安全） |
|------|--------------|---------------------|-------------------|
| TWII RSI | > 90 | 80–90 | < 80 |
| VIX | > 25 | 20–25 | < 20 |
| WTI 原油 | > 100 | 90–100 | < 90 |
| 個股 RSI | > 80 | 70–80 | < 70 |

### 核心邏輯修正（相對於舊版 TWII RSI > 85 規定）

| 舊邏輯 | 新邏輯 |
|--------|--------|
| TWII RSI > 85 → 直接禁買美股 | TWII RSI 只是預警信號之一，非單一判定 |
| 一刀切 | 分標的、分層評估 |
| 忽略 VIX | **VIX 是核心指標**（系統性風險） |
| 忽略原油 | 原油影響 Fed 降息預期，連動科技股估值 |
| 忽略個股 RSI | **個股 RSI > 80 才是進場紀律核心問題** |

**台股過熱時的正確處置：** 預警 TSM ADR 連動風險，絕非直接禁買美股。
