## 一、團隊分組標準

### 入選條件
| 團隊 | MA策略 | RSI進場 | ATR停損 | 持有天數 | 勝率門檻 |
|:----:|:------:|:-------:|:-------:|:--------:|:---------:|
| **Leo** | MA20>MA60 | RSI<60 | 2.5x | 5天 | >60% |
| **Nana** | MA20>MA60 | RSI<35-50 | 1.5-2.5x | 5-10天 | 50-65% |
| **Hunter** | 替代策略 | 消息面/動能 | 2.5-3x | 5-10天 | <50% |

### 升降標準
- **Leo → Nana**：勝率降至 <55%，持續2個月
- **Nana → Leo**：勝率升至 >65%，持續1個月
- **Nana → Hunter**：勝率 <45% 或 均報酬 <0
- **Hunter → Nana**：勝率 >50% 且 均報酬 >0

## 二、篩選流程

```
1. 初步篩選
   └── RSI < 60 + MA20 > MA60
   
2. 進場條件
   ├── Leo：RSI < 60 + MA20 > MA60 + ATR 2.5x
   ├── Nana：RSI < 35-50 + MA20 > MA60 + ATR 1.5-2.5x
   └── Hunter：替代策略（消息面/動能/價值）

3. 停損停利
   ├── Leo：ATR 2.5x 或 MA20 跌破
   ├── Nana：ATR 2x 或 RSI > 55
   └── Hunter：依替代策略設定

4. 出場條件
   ├── RSI > 55-65（中性）
   ├── 持有 > 5-10天
   └── 觸及停損/停利價
```

## 八、勝率計算標準

```
勝率 = 獲利交易筆數 / 總交易筆數 × 100%

平均報酬 = Σ(單筆報酬率) / 總交易筆數

盈虧比(PF) = 平均獲利 / |平均虧損|

信心指數 = 勝率 × 0.4 + 平均報酬 × 0.3 + PF × 0.3
```
