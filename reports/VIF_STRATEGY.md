# VIF 策略研究報告

**日期:** 2026-04-22  
**版本:** v3.12 補充條款  
**Backtester:** 完成

---

## 研究背景

v3.12 的虧損歸因分析發現：
- **86.9% 的虧損交易**發生在 VIF < 1.5 時
- 但強制 VIF>=1.5 門檻會「錯殺」好交易

---

## 關鍵發現

### 個股 VIF 特徵差異

| 股票 | VIF>=1.5 效果 | 建議 |
|:-----|:---------------|:-----|
| **3017** | ❌ 勝率下降 17.3% | VIF_Min = 1.2 |
| **3665** | ✅ 勝率提升 7.7% | VIF_Min = 1.5 |
| **2345** | ⚠️ 中性 | VIF_Min = 1.2 |

### 原因分析

| 股票 | 股性特徵 | VIF 敏感度 |
|:-----|:---------|:------------|
| 3017 | 活潑、波動大 | 低（低VIF也能漲） |
| 3665 | 穩定、趨勢明確 | 高（需要量能確認） |

---

## 策略調整

### 舊策略（已廢除）

```python
if vif < 1.5:
    return False  # 一票否決
```

### 新策略（Warning > Forbidden）

```python
if vif < stock_vif_min:
    send_warning(f"⚠️ [量能偏低] VIF={vif:.2f}")
    # 仍可進場，但縮小部位或分批買入
```

---

## VIF 分級標準

| 等級 | VIF 範圍 | 標籤 | 操作建議 |
|:-----|:---------|:-----|:---------|
| 🟢 正常 | VIF >= VIF_Warning | 無 | 正常執行 |
| ⚠️ 警告 | VIF_Warning > VIF >= VIF_Min | 量能偏低 | 縮小部位 50% |
| 🔴 高風險 | VIF < VIF_Min | 禁止進場 | 觀望 |

---

## watchlist.json VIF 配置

### 全域設定

```json
"vifGlobal": {
    "default_vif_min": 1.2,
    "default_vif_warning": 1.5,
    "strategy": "warning_not_forbidden"
}
```

### 個股設定

| 股票 | vif_min | vif_warning | 說明 |
|:-----|:--------|:------------|:-----|
| 3017 | 1.2 | 1.5 | 股性活潑 |
| 3665 | 1.5 | 2.0 | 需要量能確認 |
| 2345 | 1.2 | 1.5 | 標準設定 |
| 0050 | 1.0 | 1.2 | ETF 可較低 |

---

## 實作方式

### 進場決策流程

```
進場信號觸發
    ↓
檢查 VIF
    ↓
VIF >= VIF_Warning → 🟢 正常進場
    ↓
VIF_Warning > VIF >= VIF_Min → ⚠️ 發送警告 + 縮小部位
    ↓
VIF < VIF_Min → 🔴 高風險 → 觀望
```

---

## 結論

1. **VIF 是「警示」而非「禁止」**
2. **個股有不同的 VIF 容忍度**
3. **維持 v3.12 核心邏輯不變**
