## 一、宏觀背景：AI 資料中心引爆伺服器/儲存超級週期

2026-Q1 是伺服器與儲存產業的爆發起點：

| 驅動因子 | 說明 |
|---------|------|
| AI 基礎建設支出 | Microsoft / Google / Meta / Amazon 四巨頭 2026 年資本支出合計破 $3000 億 |
| 伺服器需求暴漲 | AI GPU 伺服器單價 $200,000-$1,000,000，訂單排隊到 2027 |
| 儲存容量倍增 | LLM 訓練需要 10-100PB 資料湖，帶動 NAND/HDD 需求 |
| HBM 記憶體短缺 | SK Hynix / Samsung HBM3e 供給吃緊，報價季增 90-95% |
| 台廠供應鏈業績爆發 | 技嘉 Q1 營收季增 18%、緯穎持續成長 |

## 八、關鍵結論

1. **記憶體超級循環**：上游 SK Hynix/Samsung/Micron 赚饱，营业益率 40-71%，但中下游 Dell/WDC 毛利改善明显，雲端存儲需求爆棚。

2. **伺服器 vs 儲存**：伺服器（Dell/技嘉）靠 AI 訂單能見度，儲存（WDC/威剛）靠 NAND/HDD 報價上漲，兩者同時間爆發，歷史罕見。

3. **台灣伺服器供應鏈**：技嘉 Q1 營收季增 18%、華碩年增 4-5x 伺服器動能，AI 伺服器滑軌大廠毛利率 77.74%，都是驗證 AI 超級週期的關鍵指標。

4. **SMCI 的矛盾**：營收低預期 $2B 但市場原諒，顯示 AI 伺服器供需極度吃緊。投資人願意容忍營收缺口，只要 EPS 超過預期且毛利率修復中。
