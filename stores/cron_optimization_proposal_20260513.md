## ⚡ 四、改善方案：重點修復

### 立即執行（4項）

**① 延長超時（7個job）**
```
模擬倉開盤前：300s → 420s（實際跑了123s但超時）
每週中度蒸餾：180s → 240s（跑了183s）
US Margin分析：120s → 180s（跑了123s）
美股策略分析：150s → 180s（跑了171s）
晚上8點提醒：30s → 60s（跑了46s）
Tina風控檢查：200s → 260s（跑了195s）
Phase2大師邏輯：600s → 600s（跑了600s，但超時）→ 保持，改為分階段
```

**② 合併重複Job（3對）**
- `Ray天條固化` + `建立ray-v3.5模型` → 合併至 `Ray天條固化`
- `Ray歷史蒸餾` + `Phase1蒸餾` → 合併至 `Ray歷史蒸餾`
- `Phase2大師邏輯` + `Ray大師人格對齊` → 合併至 `Phase2`

**③ 重新排程時段衝突**
```
09:00 → 09:10 模擬倉開盤前（避開Nana）
14:00 → 14:30 美股策略分析（避開蒸餾洪流）
17:00 → 17:30 Phase2（避開蒸餾）
18:00 → 19:30 每週中度蒸餾（避開US Margin）
20:00 → 20:15 晚上提醒
```

**④ 解決Gateway Restart中斷**
- 將Phase1/天條固化改為`sessionTarget: main`（同步執行，避免被重啟打斷）
- 或加入中斷恢復邏輯（job內檢查上次狀態）
