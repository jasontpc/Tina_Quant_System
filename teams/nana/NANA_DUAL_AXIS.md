# Nana 雙軸股票池系統 v2

## 系統架構

Nana 股票池採用「等級軸 × 產業軸」雙軸分類系統。

```
teams/nana/
├── tiers/                    # 等級軸（按 WR 勝率分層）
│   ├── tier1/               # WR ≥ 70%，高勝率短線
│   ├── tier2/               # WR 60-69%，中勝率短線
│   └── tier3/               # WR < 60%，防禦型長線
└── sectors/                 # 產業軸（按產業特性分類）
    ├── ai_hvdc/            # AI/IC/散熱/HVDC 概念股
    ├── finance/             # 金融股
    ├── defense/            # 防御型（水泥/鋼鐵）
    └── general/             # 一般電子/被動元件
```
