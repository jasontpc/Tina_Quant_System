## 🎯 評分權重系統

### 總分 = 法人分(40) + 技術分(35) + 趨勢分(25)

| 因素 | 權重 | 評分標準 |
|:-----|:----:|:---------|
| **法人分** | 40% | |
| ├ 外資連續 | | 0天=0, 1天=10, 2天=15, 3天=40, 4天=50, 6天=60, 11+天=20 |
| ├ 投信連續 | | 同上 |
| └ 合力加成 | | 雙資>=3天 +10 |
| **技術分** | 35% | |
| ├ RSI | | 50-70=15, 30-50=10, 其他=5 |
| ├ Bias | | -2~3%=15, 3~6%=10, >10%=0, <-5%=5 |
| └ ATR% | | >=0.5%=10, >=0.3%=5, <0.3%=0 |
| **趨勢分** | 25% | |
| ├ MA20>MA60 | | 是=15, 否=0 |
| ├ 價格動能 | | 20日漲幅 >0%=10, >5%=15 |
| └ 相對大盤 | | 優於大盤=10, 落後=-5 |

## 🔄 動態條件系統

### 市場狀態判斷

| 狀態 | 條件 | 策略調整 |
|:-----|:-----|:---------|
| **多頭** | TWII > MA20, RSI 50-70 | 順勢進場, 提高倉位 |
| **盤整** | TWII 區間震盪 | 均值回歸, 區間操作 |
| **空頭** | TWII < MA20, RSI <50 | 保守進場, 降低倉位 |
| **過熱** | RSI > 80, Bias > 10 | 只出不進 |
| **恐慌** | VIX > 30 | 完全觀望 |

### 行業特性對應

| 行業 | 特性 | 對應策略 |
|:-----|:-----|:---------|
| 半導體 | 高波動 | ATR>=0.5%, 嚴格停損 |
| AI伺服器 | 動能強 | 順勢, 移動停損 |
| 金控 | 低波動 | ATR>=0.3%, 高股息濾網 |
| ETF | 穩健 | 大盤同步, 低頻交易 |
