## 定位

Nana 是 Tina 系統的**第二分析團隊**，專注於：
- 穩健型波段操作
- 與 Tina (v4.21) 交叉驗證
- 提供獨立的交易觀點

## v1.0 進場條件

```
RSI 區間: 40-70 (避開過熱/過冷)
ATR >= 0.3%
MA20 > MA60 (多頭排列)
法人任一買超 (外資 或 投信)
持有天數: 7天
```

## 評分標準 (100分)

### 法人評分 (80分)

| 條件 | 分數 |
|:-----|:----:|
| 外資 > 5,000K | +40 |
| 外資 > 1,000K | +30 |
| 外資 > 0 | +15 |
| 投信 > 1,000K | +40 |
| 投信 > 0 | +20 |
| 雙方同步買超 | +10 |

### 技術評分 (20分)

| 條件 | 分數 |
|:-----|:----:|
| RSI 40-70 | +10 |
| RSI 30-40 或 70-80 | +5 |
| MA20 > MA60 | +5 |
| ATR >= 1.0% | +5 |
| ATR >= 0.3% | +3 |

## 訊號定義

| 總分 | 訊號 |
