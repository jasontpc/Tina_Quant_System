## 🎯 Nana v6.4 波段策略

### 進場條件
```
RSI(12) 在 30-45（超賣）
近5日動量 > 3%（強回調）
ADX > 18（趨勢明確）
評分 >= 32
MA60 多頭排列
```

### 停利/停損
```
停利：3x ATR 或 MA20 跌破
停損：1.5x ATR
持有：最多 7 天
```

### 適用場景
- 📈 多頭市場的回檔買點
- 📊 震盪市場的區間操作
- ⚠️ 不適合空頭市場

### 優勢
- 進場成本低（超賣）
- 停損幅度小（1.5x ATR）
- 篩選嚴格，減少假訊號

### 劣勢
- 等待時間長（RSI 30-45 不常見）
- 可能錯過強漲股
